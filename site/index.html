Qpackt - Web Deployment and Monitoring

Qpackt is a web server designed to make your life easier:

1. Serve Multiple Versions of Your Website:

  • Visitors are assigned a version on the first request.
  • Consecutive requests are served from the selected version.
  • This enables:
    • a) A/B testing
    • b) Loading multiple versions based on specified URL parameters
    • c) Deploying new versions while preserving old sessions

2. Basic, GDPR-Compliant Analytics:

Includes metrics like bounce rate and visit count, collected without cookies.

3. Auto-Fetch SSL Certificates:

Automatically fetch (and renew) SSL certificates from LetsEncrypt.

4. GUI Configuration:

Provides a graphical user interface for ease of use.

5. Open Source and Free:

Fully open source and free to use. Source code.

Qpackt is not yet production ready (see roadmap) but is able to serve itself (if you can read this).
