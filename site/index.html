Qpackt - Modern Web Server

Qpackt is a webserver for 21st century:
  • Fast and reliable (built in Rust)
  • Non-invasive analytics (http-based and custom events)
  • A/B testing
  • SSL certificate management
  • Open-source (GitHub)
  • Web-based admin's console
  • ... and more
